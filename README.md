# 🛒 SmartCart Customer Segmentation System

## Overview
SmartCart Customer Segmentation System is an Unsupervised Machine Learning project that segments e-commerce customers into different groups using the K-Means Clustering algorithm.

The system analyzes customer purchasing behavior, demographics, spending patterns, and engagement metrics to identify meaningful customer segments for targeted marketing and business decision-making.

---

## Features

✅ Customer Segmentation using K-Means Clustering

✅ Data Preprocessing and Feature Engineering

✅ One-Hot Encoding for Categorical Features

✅ Feature Scaling using StandardScaler

✅ Dimensionality Reduction using PCA

✅ Interactive Streamlit Web Application

✅ Real-Time Customer Segment Prediction
